# FIVB World Ranking Globe

Interactive 3D globe visualization of FIVB Senior World Rankings for volleyball.

## Features

- Interactive 3D globe with D3.js orthographic projection
- Live data from official Volleyball World API
- Men's and Women's rankings toggle
- Country cards with rank, points, recent matches, and points trend chart
- Hover tooltips on sparkline to view historical points

## Tech Stack

- D3.js v7
- TopoJSON
- Tailwind CSS
- Vanilla JavaScript

## Usage

`ash
git clone https://github.com/DeGrozer/volleyball-world-rankings.git
cd volleyball-world-rankings/globe-lineart
npx serve .
`

## Data Source

Rankings fetched from [Volleyball World](https://en.volleyballworld.com/) official API.
